Many company seeks to expand their domain by having branches located in different cities/countries. Unity have a branch in Japan with the branch name Unity Technologies Japan. They created a mascot character for the Unity brand; a character named Unity, or Unity-chan with the honorifics. The creators sought to distribute the Unity-chan asset freely to developers using Unity. The creation of Unity-chan led to an assorted rise of merchandise, events and games, which ultimately is a marketting success.
